This is an AI-powered application designed to provide detailed information and insights into various scientific revolutions throughout history.

## Tech Stack
- **Frontend:** React.js (Vite) / HTML5 & CSS3
- **Backend:** Node.js / Python
- **AI Integration:** Google Gemini API
